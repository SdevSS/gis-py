# 🌲 Aplikasi Mapping Peta Deforestasi 🌲

![image](https://i.imgur.com/FJGzMZ7.png)

A lightweight, accessible alternative to heavyweight GIS software for simulating
**deforestation, edge effects, carbon-stock valuation, and habitat connectivity**,
built as a decision-support tool for endemic-species of flora and fauna conservation (e.g. Papua).

Draw or import a landscape on a 20×20 hectare grid, run the simulation, and the
dashboard instantly reports intact forest, edge degradation, carbon value (IDR),
and whether the habitat's biological corridors are still intact.
